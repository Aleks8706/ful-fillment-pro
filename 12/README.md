# Cosmetic WMS Pro

Cosmetic WMS Pro — это одностраничная WMS-система для склада косметики с поддержкой партий, сроков годности, FEFO-отбора, ролей пользователей, инвентаря и заказов.

## Возможности

- Управление товарами.
- Управление локациями склада.
- Приёмка партий с указанием срока годности.
- Отгрузка по FEFO.
- Трансфер партий между локациями.
- Создание и обработка заказов.
- Поиск товаров.
- Отчёты по низкому остатку и истекающим срокам.
- Сохранение данных в localStorage.
- Экспорт данных в JSON.

## Структура файлов

- `index.html` — разметка интерфейса.
- `styles.css` — оформление.
- `app.js` — логика приложения.

## Запуск

1. Скопируйте все файлы в одну папку.
2. Откройте `index.html` в браузере.
3. Для проверки работы используйте кнопку `Демо`.

## Учёт косметики

Для косметики используется FEFO — сначала отгружаются партии с ближайшим сроком годности. Это помогает уменьшить списания и потери.

## Пользователи

Встроенные роли для демо-версии:

- `admin / 1234`
- `manager / 1234`
- `picker / 1234`
- `viewer / 1234`

## Хранение данных

Данные сохраняются в `localStorage` текущего браузера.  
Это удобно для локальной демо-версии, но для реального склада рекомендуется backend и серверная авторизация.

## Формат партии

Каждая партия товара хранит:

- товар,
- локацию,
- номер партии,
- количество,
- дату поступления,
- срок годности,
- статус.

## Пример использования

1. Войти в систему под пользователем.
2. Добавить товар.
3. Добавить локацию.
4. Принять партию с датой годности.
5. Создать заказ.
6. Отгрузить товар по FEFO.
